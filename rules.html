Wiki Race

  • The goal is to find a path from one Wikipedia article to another by clicking through linked articles.
  • Click randomize for a Start Point and End Point (for an easier game, click randomize until you find Start/End Points you feel comfortable with)
  • Once you have selected a Start Point and an End Point, the race begins and the first person to reach the End Point's Wikipedia page WINS!
  • Hint: using the Find function (ctrl-F) can help you manuever through the page (make sure all players agree it's ok to use it!)

A complete list of rules can be found on Wikipedia
